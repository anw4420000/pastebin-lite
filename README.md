# Paste App (Next.js)

This is a **Next.js** project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app). It allows creating and retrieving pastes via API endpoints, and includes a health check for the database.

---

## Getting Started

First, install dependencies and start the development server:

```bash
npm install
npm run dev
# or
yarn install && yarn dev
# or
pnpm install && pnpm dev
